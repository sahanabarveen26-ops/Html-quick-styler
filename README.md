HTML Quick-Styler

AI-Powered HTML Page Generator

📌 Project Overview

HTML Quick-Styler is an AI-powered tool that helps users generate complete HTML web pages with modern styling and responsive design.
The system analyzes user input and automatically generates structured HTML and CSS code.

🚀 Features

- Automatic HTML page generation
- Modern CSS styling
- Responsive web design
- Pre-built section templates
- Color palette generator
- Clean and structured code output

🛠 Technologies Used

- Python
- HTML
- CSS
- Google Colab
- IBM Granite Model

📂 Project Structure

This project contains the following files:

- Project Notebook – Main implementation of the project (".ipynb" file)
- Output Screenshot – Image showing the generated HTML page output
- Output Screen Recording – Video recording demonstrating the project execution with voice explanation
- Project Document – Detailed explanation of the project including design and workflow

🎯 Purpose

The purpose of this project is to simplify web development by automatically generating HTML pages with professional design using AI.
